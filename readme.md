## ⚡ Quick Download & Installation Guide

Crystal AI uses a **Rolling Release** continuous pipeline. You do not need a computer or a file extraction utility to install updates. Android handles the deployment natively.

### 1. Grab the Package
Look at the right-hand column of this page (or tap the menu bar on mobile) and select **Releases**. Tap on the latest build labeled **`Crystal AI (Latest Build)`**.

### 2. Download the Binary
Under the **Assets** dropdown menu of the latest release, click directly on:

app-release-signed.apk
3. Native Deployment
Open your device's file manager or download history and tap the downloaded .apk file.
If your system displays a security prompt, select Settings and flip on "Allow installation from this source".
Tap Install (or Update if you are overwriting an older version). Crystal AI will seamlessly configure itself and launch!

🛠️ Installation Troubleshooting
🛑 "App Not Installed" Error
This occurs if there is a signature conflict between an old testing package and the new deployment build.
The Fix: Completely uninstall your current version of Crystal AI from your phone app drawer, clear your active download history cache, download the new .apk file from this release repository, and rerun the package installer.
⚠️ "Blocked by Play Protect" Warning
Because this package is compiled directly from a private GitHub automated cloud server pipeline rather than uploaded onto the Google Play Store, Android will occasionally warn you that it's an unrecognized developer package.
The Fix: Tap "More Details" inside the warning pop-up element, then select "Install Anyway" to complete setup.
